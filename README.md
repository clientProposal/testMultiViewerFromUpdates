# Steps to use

1. Grab your trial key from the [Apryse Downloads page](https://dev.apryse.com/). If it doesn't appear, create an account and log in, or log in if you already have one.

2. Make a .env file and paste it like this:
VITE_PDFTRONKEY=your-key

3. npm install, npm run dev. That's it! 

If you keep selecting one window and then the other, you'll receive prompts to observe improvements to the multiview mode.
